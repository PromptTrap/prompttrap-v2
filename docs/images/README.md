# PromptTrap Screenshots

This directory contains screenshots for documentation.

## To add screenshots:

1. Run the demo: `npm run demo`
2. Start the dashboard: `npm run dashboard`
3. Open http://127.0.0.1:9099
4. Take screenshots of:
   - Dashboard overview showing all panels
   - Activity feed with DLP findings
   - DLP summary panel
   - Tool usage chart
   - Session timeline

Save as:
- `dashboard-overview.png` - Full dashboard view
- `activity-feed.png` - Activity feed detail
- `dlp-findings.png` - DLP summary detail
- `tool-chart.png` - Tool usage visualization

## Image guidelines:

- Use a 16:9 or 16:10 aspect ratio
- Minimum width: 1280px
- Format: PNG with transparency where possible
- Dark theme screenshots preferred
- Highlight DLP findings and policy actions
